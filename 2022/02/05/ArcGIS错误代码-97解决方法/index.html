ArcGIS错误代码-97解决方法

Shixi 2022-02-05 22:36:46
Categories： ArcGIS Tags：

error code=-97表明出现了许可错误，解决问题的关键在于修改service.txt文件。

在修改之前，你需要关闭正在运行的许可证服务。打开ArcGIS License Server Administrator，选择停止许可服务，同时在任务管理器中确保它已经关闭。如果无法停止，可以尝试通过控制面板中的服务项手动关闭ArcGIS License Manager。

接着，我们需要准备一个用来替换原文件的新service.txt。(该文件已有许多公开版本，本篇文章不提供文件来源。)

在获得文件之后，找到你安装License的途径，并替换原有的txt文件。

最后，重新启动许可证服务，问题解决。
